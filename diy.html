Starting Color: Number of Colors:

Welcome to the DIY Generative Art Page. To get started choose a starting color hue number and how many colors you want, the higher the number the less variance in color as the color hue will increase by less. Once you have selected a color and number of colors click the start drawing button. If at anytime you want to stop drawing click the stop drawing button. You can download your creation by clicking Download as PNG. You can clear the canvas at any time with the clear canvas button. This message will disappear once you click start drawing.
